# Dice Game

Ye ek simple dice game hai jisme aap coins laga ke khel sakte ho.  
Game HTML, CSS, aur JavaScript se bana hai.

## Features  
- Coins betting system  
- Dice roll randomization  
- Coins localStorage me save hote hain  

## Kaise khele  
1. Apne coins bet karo  
2. Dice roll karo  
3. Jeet ya haar ke hisaab se coins badhenge ya ghatenge  

Enjoy karo aur feedback do!
